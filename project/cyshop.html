CyShop

Online Shopping & Ordering System, Better Deal

CyShop is a Spring and Hibernate based online Shopping and Ordering system. It employed Spring framework to build a web applicaiton for users to shop and order items online.

Technical Details

The web application CyShop is built based on SPring MVC to support item search and listing. Concepts such as dependency injection, inversion of control, and RESTful APIs are also utilized in development.

In terms of the website security, we implemented security workflow via in-memory and JDBC authentication provided by Spring Security. Hibernate is also used to provide better support of database operations. A Spring Web Flow is carefully designed and developed to support item ordering.
