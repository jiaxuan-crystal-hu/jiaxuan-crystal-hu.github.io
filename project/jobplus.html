Job+

A Job Recommendation Web Service Development

Job+ is an interactive web page for users to search and apply for positions. It utilizes users' favorite records to provide personalized position recommendation.

Technical Details

Job+ employs three Java servlets with RESTful APIs to handle HTTP requests and responses. It retrieves job position information through Github API. It connects to MySQL database on Amazon Relational Database (RDS) to store position data from Github API.

We also utilizes MonkeyLearn API to extract keywords from description of positions to present a precise and inclusive search result for users. The conent-based recommendation algorithm helps to implement job recommendation based on user's favorite record and searching results.
